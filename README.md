## ✅ Tasks Completed

### Task 1 — Web Scraping
- Scraped IMDB Top 250 movies using Python
- Libraries: Requests, BeautifulSoup, Pandas
- Output: 250 movies × 9 features CSV

### Task 2 — Exploratory Data Analysis
- Analyzed ratings, genres, directors, decades
- Found key patterns and insights
- Libraries: Pandas, Numpy, Matplotlib, Seaborn

### Task 3 — Data Visualization
- Built 6 professional charts
- IMDB-themed color palette
- Libraries: Matplotlib, Seaborn, Scipy

---

## 🔍 Key Insights
- ⭐ Average IMDB rating: **8.31**
- 🏆 Highest rated: **The Shawshank Redemption (9.3)**
- 🎬 Top directors: **Nolan, Kubrick, Scorsese, Kurosawa**
- 🎭 Most common genre: **Drama (27%)**
- 📅 Most movies from: **2000s decade**
